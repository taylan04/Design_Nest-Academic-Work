Design Nest

Estou planejando desenvolver um projeto de interface para um
site chamado Design Nest, uma marca focada em oferecer serviços de design
gráfico. Utilizando HTML, CSS e JavaScript, meu objetivo é criar uma
plataforma atraente e funcional, que represente a identidade visual da marca,
proporcionando uma experiência de usuário intuitiva e moderna. A ideia é que o
site possa simular características de um site verdadeiramente ativo.


Você pode acompanhar todo o processo do projeto desde a sua criação até a sua
conclusão através do meu GitHub.


É o meu primeiro (de muitos, amém) site nessa escala. Sei que para muitos não
é nada demais, mas eu estou muito feliz de ter chegado até aqui. Sou extremamente grato
a todos os profesores e colegas que me ajudaram a adquirir aprendizado suficiente para esse projeto.


Não vejo a hora de poder implementar nesse site outras tecnologias como React, APIs, Responsividade e etc.
